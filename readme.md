# 🚀 Demo Página Web para Negocios (Ventas por WhatsApp)

Este proyecto es un ejemplo real de una página web diseñada para ayudar a negocios a vender de forma más organizada y rápida utilizando WhatsApp.

---

## 🎯 Objetivo

Mostrar cómo un negocio puede:

- Exhibir sus productos
- Recibir pedidos directos
- Reducir preguntas innecesarias
- Aumentar ventas

Todo sin necesidad de aplicaciones complejas.

---

## 💡 ¿Qué incluye esta versión? (Plan 200K)

👉 Catálogo básico de productos  
👉 Imágenes y precios visibles  
👉 Botón directo a WhatsApp  
👉 Diseño responsive (celular y computador)  
👉 Entrega rápida (estructura simple y funcional)

---

## 🧠 ¿Cómo funciona?

1. El cliente entra al link
2. Ve los productos disponibles
3. Hace clic en "Pedir"
4. Se abre WhatsApp con el mensaje listo

---

## 🔥 ¿Por qué funciona?

Este sistema está diseñado para tres cosas clave:

1. Reducir preguntas
2. Aumentar pedidos
3. Ahorrar tiempo

👉 El cliente ya llega con intención de compra.

---

## ⚠️ Antes vs Después

**Sin página:**

- El cliente pregunta todo
- Pierdes tiempo respondiendo
- Menos cierres

**Con esta página:**

- El cliente ve todo
- Decide más rápido
- Te escribe listo para comprar

---
